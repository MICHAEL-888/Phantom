# 灵蛇ARK (Phantom Anti-Rootkit)

一款r3层强大的Windows平台反Rootkit安全工具。

## 功能特性

- 进程管理
- 文件系统管理
- 注册表管理
- 系统服务检测
- 启动项检测

## 系统要求

- Windows 10/11 (64位)

## 使用说明

1. 以管理员身份运行程序
